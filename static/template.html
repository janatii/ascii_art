this little project capture frames from mp4 videos, convert them to an ascii art image, then stream them to the browser in realtime

this is all happening in a tiny old computer of mine, so I try to keep the captured frames to a minimum in order not to heat the room so much

https://github.com/janatii/ascii_art

source code
